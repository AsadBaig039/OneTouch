### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AsadBaig039/OneTouch.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

### Run Application

1. Open VSCode Terminal
2. Run the Command
   ```sh
   react-native run-android
   ```
